# BeamNG.Drive in Python 2D

A 2D soft-body physics simulation inspired by BeamNG.Drive, built with Python, Pygame, and Pymunk.

## Features
* **Soft-Body Physics:** Nodes and beams system for realistic car deformation.
* **Suspension System:** Damped springs for smooth driving over obstacles.
* **Real-time Camera:** Smooth camera tracking following the vehicle.
* **Cross-platform:** Works on PC and Android (via Pydroid 3).

## Installation
Make sure you have Python installed, then run:
